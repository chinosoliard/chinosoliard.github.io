Telegram has become, in the last years, in a widely used application, especially among IT guys. The messaging service has a web app, that allow us using the service without having a smartphone. I decided to try a desktop app so I do not have to log in every time I open Firefox (which I have configured so I do not save anything).
In this post, I'll tell you a about Telegram, the differences with WhatsApp, what IT guys do with it, and how to install and use Cutegram (a client app for Telegram).

About Telegram...

What is Telegram?

Telegram is a messaging service, like WhatsApp, to which all it have to envy are calls and video calls, but it is more focused on speed and security.
The client is Free Software (GPL), which has made the service widely used among Free Software activists, not so the server, whose license is private, and its procedures are not public either; It should be noted that Telegram works based on the cloud.

Advantages about WhatsApp

  • Cloud
    Being Telegram a cloud-based service, messages and files are available on any device we use Telegram, in the web application or desktop applications.

    You don't need to have your phone turned on and / or connected to the internet to use Telegram
