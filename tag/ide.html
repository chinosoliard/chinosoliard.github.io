Multiples instalaciones de Eclipse IDE en el mismo sistema

###############
Date = 2014-04-05
Category = Fedora
Tags = c++ cpp development eclipse IDE Fedora GNU/Linux jee Php
Lang = es
Translations: none
###############

Creo que no soy el único que piensa que Eclipse IDE se torna pesado a medida que le agregamos plugins y demás, y, si bien tiene la facilidad de cambiar entre áreas de trabajo (Workspaces), personalmente prefiero separarlos del todo.

En mi caso en particular, trabajo con Java, veo algunas cosas en C y estoy haciendo un curso de PHP, entonces, decidí tener 3 Eclipse distintos en mis equipos (obviamente, uno para cada lenguaje), pero puede usarse con distintos fines, por ejemplo, uno para Java y SWING y otro para Java EE, etc.
